## 📊 Project Overview
This project presents an interactive **Sales Dashboard** built in **Power BI** using the SuperStore dataset.  
The dashboard helps analyze **sales performance, profit trends, customer segments, and shipping modes** to support business decision-making.

## 🎯 Objectives
- Analyze overall **Sales and Profit**
- Identify **top-performing categories and sub-categories**
- Track **monthly sales and profit trends**
- Understand **customer segments**
- Compare **shipping modes and payment methods**
- Visualize **state-wise sales performance**

## 📈 Dashboard Features
The dashboard includes the following visualizations:

- **Total Sales, Profit, Orders, and Ship Days KPIs**
- **Sales by Category**
- **Sales by Sub-Category**
- **Sales by Ship Mode**
- **Sales by Payment Mode**
- **Sales by Customer Segment**
- **Monthly Sales Trend**
- **Monthly Profit Trend**
- **State-wise Sales Map**

## 📊 Key Insights
- Consumer segment contributes the highest sales.
- Standard Class shipping is the most commonly used shipping mode.
- Technology category generates the highest revenue.
- Monthly trends show fluctuations in both sales and profit.
